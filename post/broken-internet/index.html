Broken Internet?

Mon, Nov 9, 2015

There is something funny going on with upcoming generic top level domains (gTLDs), they seem to be looked up in a strange manner (at least on latest Linux). For example:ping chrome orping nexus returns 127.0.53.53.While existing official gTLDs don’t (ping dental returns “unknown host” as expected). I first thought it was a network misconfiguration, but as I am not the only one to notice this, it’s likely a genuine internet issue.Strange times.
