Jython instead of Ant: sample code

Jul 16, 2005 · 2 minute read · Comments

Here is an example of how to use jython to deploy a web application under tomcat. The application is supposed here to be WAR packaged in a ‘web’ directory. Libraries used for the web-app are already in web/WEB-INF/lib, classes are compiled automatically by Eclipse under web/WEB-INF/classes, and resources are copied automatically by Eclipse. If necessary, it is very easy to script those actions in the python script, with very little code.The longest code is the TreeCopier class, that should probably be part of any python make script. The reload method could be reworked to provide a generic method for reloading a tomcat web application.
